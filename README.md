Quick-parcel-api-node by quentinmdl

# Description 

Basic entity api with Node + Fastify + Mongoose

# Methods : 

GET, POST, PUT, DELETE

# Routes: 

- post
- getAlls
- search (option: ?term=)
- getById/:id
- updateById/:id
- deleteById/:id

# Started

Configure and rename all files :

    - Env
    - server.js
    - Controller
    - Models
    - Routes
    - Validations

# Commands

Install the node modules 

```console
    npm install
```

```console
    npm install
```

Then run the project with it

```console
    npm run dev
```
